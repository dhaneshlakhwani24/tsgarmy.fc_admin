# TSG Army Esports Platform

Hi, myself **Dhanesh Lakhwani**. I am a developer, and this website is made in collaboration with **[@tsgarmy.fc](https://instagram.com/tsgarmy.fc)**.

This is a full-stack esports project with a public website, a protected admin dashboard, and a backend API for players, schedules, achievements, and feedback.

## Quick Overview

- Public website for team showcase and updates
- Admin panel for content and operations management
- Secure backend with authentication and role-based access
